# Subscription Churn Prediction

Machine learning project focused on predicting user churn
with an emphasis on product insights and business interpretation.

---

## Problem

Identify users with a high probability of churn and understand
the key drivers behind churn behavior.

---

## Approach

1. Exploratory data analysis
2. Feature interpretation with a product mindset
3. Baseline ML models:
   - Logistic Regression
   - Random Forest
4. Model evaluation using business-relevant metrics:
   - ROC-AUC
   - Precision
   - Recall
5. Analysis of churn drivers
6. Product recommendations for retention strategies

---

## Key Focus

- Interpretability over model complexity
- Metric selection driven by business goals
- Actionable insights for product teams

---

## Tech stack

- Python
- Pandas, NumPy
- scikit-learn
- Matplotlib
- Jupyter Notebook
